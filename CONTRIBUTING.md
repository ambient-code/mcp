# Contributing

## Setup

```bash
git clone https://github.com/jeremyeder/mcp.git
cd mcp
bun install
./scripts/setup-hooks.sh
```

## Development

```bash
./scripts/validation/check.sh
```

## Commits

Use [conventional commits](https://www.conventionalcommits.org/):
- `feat: add new feature`
- `fix: resolve bug`
- `docs: update readme`
